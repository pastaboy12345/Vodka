# Vodka
A project aiming to create a way to run Android apps, natively, on Linux.

## Q&A

Q. What is Vodka?
A. Vodka is a project that aims to create a way to run Android apps, natively, on Linux...

Q. Is it an emulator?
A. No, it is not an emulator. Instead, like "debtap" on Arch Linux, it extracts given "APK" file and converts it into either a .deb file, .rpm file, or a .pkg.tar.zst file, depending on the user's preference.

Q. Is it very good and can run most apps?
A. Not either. Like Darling, Android is very complicated and as such, Vodka can't run apps that require the ART or Google Play Services.

Q. Is it a developer tool?
A. Yes, it can be used for developing Android apps (beta).

Q. Is it a replacement for Waydroid?
A. No, it is not a replacement for Waydroid. Waydroid is a container-based Android system that runs Android in a container, while Vodka is a tool that converts Android apps into native Linux apps.

Q. Is it a replacement for Wine?
A. No, it is not a replacement for Wine. Wine is for Windows apps on Linux, while Vodka is for Android apps on Linux.

Q. Is it a replacement for Anbox?
A. No, it is not, however if you are using Anbox, you should probably switch to Waydroid.
